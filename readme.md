## Commands run in order

Install nestjs

```zsh
npm i -g @nestjs/cli
```

Init nestjs project

```zsh
nest new project-name
```

Generate a controller

```zsh
nest g controller items
```

Generate a service

```zsh
nest g service items
```

Start the server

```zsh
pnpm run start:dev
```
